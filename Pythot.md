# Синтаксис Python

### range() переберает числ
    range(stop) 
    range(start, stop[, step])
    
    range(5) -> 0 1 2 3 4
    range(1, 5) -> 1 2 3 4
    range(1, 5, 2) -> 1 3
Источник: https://pythonstart.ru/osnovy/funktsiya-range-python

### len() возвращает количество элементов (длину) в объекте.

    x = "привет"
    len(x) == 6
